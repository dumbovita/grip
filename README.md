# GRIP - Firefox Extension

Get Right-click Images Properly: save images as PNG, JPG, or WebP.

## Requirements

- **Node.js**: v18 or higher
- **npm**: v9 or higher (comes with Node.js)
- **Operating System**: Windows, macOS, or Linux

## Installation

1. Install Node.js from [nodejs.org](https://nodejs.org/)
2. Clone this repository
3. Install dependencies:
   ```bash
   npm install
   ```

## Build Instructions

### Development

```bash
# Development mode (Firefox)
npm run dev:firefox

# Development mode (Chrome)
npm run dev
```

### Production Build

```bash
# Build for Firefox
npm run build:firefox

# Build for Chrome
npm run build
```

### Create ZIP for Submission

```bash
# Create Firefox ZIP
npm run zip:firefox

# Create Chrome ZIP
npm run zip
```

### Run Tests

```bash
npm run test
```

## Build Script

All build steps are automated through npm scripts defined in `package.json`. The build process:

1. Installs all dependencies via `npm install`
2. Compiles TypeScript using WXT framework
3. Outputs the extension to the `.output/` directory

## Project Structure

- `entrypoints/` - Extension entry points (background, content scripts, popups)
- `src/` - Source TypeScript files
- `public/` - Static assets
- `.output/` - Build output directory
- `wxt.config.ts` - WXT configuration

## Notes

- All source files are written in TypeScript and are not minified or transpiled in the repository
- Third-party libraries are managed through npm and listed in `package.json`
- The build output is generated by the WXT framework and can be reproduced exactly by running the build commands above
